# Инструкция по загрузке в Яндекс Диск (Русская версия)

Назад к индексу облаков: [docs/clouds/README.md](README.md)

English version: [yandex-disk.en.md](yandex-disk.en.md)

## Когда выбирать Яндекс Диск

Используйте Яндекс Диск, если вам нужен привычный веб-интерфейс, общие папки и простая синхронизация на macOS/Windows.

## Веб-загрузка (macOS, Windows, Linux)

1. Откройте [веб-версию Яндекс Диска](https://disk.yandex.com) и войдите в аккаунт.
2. Создайте папку назначения, например `GooglePhotos-Migration`.
3. На компьютере откройте `takeoutfix-extracted/Takeout`.
4. Перетащите всё содержимое папки `Takeout` в папку назначения в браузере.
5. Дождитесь завершения загрузки, обновите страницу и проверьте количество файлов.

Рекомендуемый скриншот: `docs/images/10_yandex_web_upload.png`.

## Синхронизация через приложение на macOS

1. Установите приложение Яндекс Диск с официального сайта Яндекса.
2. Войдите в аккаунт Яндекса.
3. Откройте локальную папку Яндекс Диска в Finder.
4. Скопируйте содержимое `takeoutfix-extracted/Takeout` в папку синхронизации.
5. Дождитесь статуса, что все файлы загружены.

## Синхронизация через приложение на Windows

1. Установите приложение Яндекс Диск с официального сайта Яндекса.
2. Войдите в аккаунт и завершите первичную настройку.
3. Откройте локальную папку Яндекс Диска в Проводнике.
4. Скопируйте содержимое `takeoutfix-extracted/Takeout` в папку синхронизации.
5. Дождитесь, пока приложение покажет завершение синхронизации.

Рекомендуемый скриншот: `docs/images/11_yandex_desktop_sync_windows.png`.

## Резервный путь для Linux (только веб)

Для нетехнической миграции на Linux используйте веб-загрузку. Это самый предсказуемый и стабильный путь на разных дистрибутивах.

## Чеклист после загрузки

1. Откройте минимум 20 случайных фото и 5 случайных видео.
2. Проверьте, что даты съёмки в ленте отображаются правильно.
3. Проверьте геометки у файлов, где они были изначально.
4. Убедитесь, что структура папок сохранена.
5. Проверьте, что в истории активности Яндекс Диска загрузка завершена.

## Частые ошибки и быстрые решения

- Ошибка: загружаются ZIP-архивы вместо обработанных файлов.
  - Решение: загружайте только `takeoutfix-extracted/Takeout`.
- Ошибка: папки «сплющиваются» при перетаскивании.
  - Решение: пересоздайте корневую папку и загружайте с сохранением структуры.
- Ошибка: локальные файлы удаляются слишком рано.
  - Решение: храните локальную копию до полной проверки.
- Ошибка: загрузка в браузере прервалась.
  - Решение: дозагрузите отсутствующие папки и снова сверьте количество файлов.
