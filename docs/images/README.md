# Screenshot Checklist and Rules

This folder stores screenshots used by README and cloud guides.

## Naming Convention

- Use `NN_short-description.png`.
- Keep two-digit prefixes (`01`, `02`, ...).
- Use lowercase letters, digits, and hyphens only.

## Quality Rules

1. PNG format only.
2. English UI for cloud service interfaces.
3. Blur personal data (names, email, account IDs, private folder names).
4. Use minimal callouts (rectangles/arrows) only on key UI actions.
5. Keep consistent size where possible (for example 1440px width).
6. Do not crop out crucial context like upload progress or completion status.
